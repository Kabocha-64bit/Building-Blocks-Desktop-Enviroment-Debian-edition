# Building-Blocks-Desktop-Enviroment
BBDE (Building Blocks Desktop Enviroment) Is a DE completly made by the user. The Installation file (BBDE-OPIN.py) gives you choices for Window manager, Panel, and Wallpaper manager. Have fun making a DE! (BBDE comes with Rofi and Plank by Default. See Rofi and Plank Docs to know how to use them.)

 Show off your BBDE Setup!
I don't want to do a billion installs to show every combination,so I want to see yours! 

If you've built a cool desktop using BBDE, please share it:
1. Take a screenshot of your desktop.
2. Open a new [Issue](https://github.com/issues) and upload your image. (Assign to me)
3. (Optional) Include your config files so others can use your setup!

I'll feature the best ones right here in the README.
