# Building-Blocks-Desktop-Enviroment
BBDE (Building Blocks Desktop Enviroment) Is a DE completly made by the user. The Installation file (BBDE-OPIN.py) gives you choices for Window manager, Panel, and Wallpaper manager. Have fun making a DE! (BBDE comes with Rofi and Plank by Default. See Rofi and Plank Docs to know how to use them.)
